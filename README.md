Personal Portfolio Website (Static)

A simple static personal portfolio webpage created using HTML and CSS. The website presents basic information such as profile details, services, and portfolio content in a clean layout.
This project focuses on demonstrating fundamental frontend structure and styling without using JavaScript or backend technologies.

Project Purpose

The purpose of this project is to practice and demonstrate:
Basic HTML page structure
CSS styling and layout design
Creating a simple static portfolio webpage

Features

Static single-page portfolio layout
Profile and introduction section
Services / work showcase section
Portfolio content section
Social media icons using Font Awesome
Simple clean UI design

Note:
This project does not include interactive functionality such as forms, JavaScript features, or backend integration.

Technologies Used

HTML5 – Structure of the webpage
CSS3 – Styling and layout
Font Awesome – Icons for social media and UI elements
